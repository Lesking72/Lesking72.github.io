Do I really need to restart after each failure?

If you get an "Out of memory" error, you can dismiss it and let the page reload to try again. If you get any other
error, restart the console through the quick menu or using the power button on the console. Don't close the dialog box,
because that usually results in a panic.

Should the screen go black when starting Mira?

Mira restarts the shell when it loads, this behavior is normal.

What's IDU mode?

Store demo mode. It automatically updates the firmware, which will leave you stuck in
IDU mode because you won't be able to access the debug settings to turn it off.
DON'T ENABLE IDU MODE OR YOU'LL BE STUCK ON THE LATEST FIRMWARE, AND IN IDU MODE.
If you enabled it and had your firmware update, the only thing you can do is put your PS4
in the back of a closet until an exploit is released for the version it updated to.
